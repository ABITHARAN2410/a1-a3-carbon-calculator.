Embodied Carbon Model for Concrete

This calculator is set up for temporary works decision-making. It compares carbon, cost and early-age compressive strength so a user can identify a mix that is lower carbon while still meeting practical strength requirements.

Scope: A1–A3 only. Transport to site, batching energy, curing energy, reinforcement and end-of-life are excluded unless added later.
Method: Quantity × carbon factor for each material, summed for the selected cube volume.
Disclaimer: This is a dissertation prototype. Replace default factors with verified project EPDs before using for real project decisions.

5. User guidance

How to use this prototype

Steps

  1. Select a concrete mix from the dropdown.
  2. Confirm cube size, number of cubes and wastage allowance.
  3. Review the material quantities, carbon total, cost and strength summary.
  4. Compare all mixes using the dashboard and comparison table.
  5. Replace default factors with verified EPD values for final dissertation calculations.

Formula used

Wet volume: (cube size ÷ 1000)³ × number of cubes × (1 + wastage %)

Material quantity: material quantity per m³ × wet volume

Embodied carbon: material quantity × A1–A3 carbon factor

Cost: material quantity × material cost factor
